## What it does

**Mongli_Agent_IA** is an autonomous on-chain intelligence agent that monitors "smart money" wallets on the Mantle Network (EVM Layer 2) in real time. It detects anomalous patterns and unusual movements using machine learning models, then writes verifiable signals directly to a Solidity smart contract on Mantle.

Every alert generated by the AI is cryptographically recorded on-chain with a confidence score and a data hash, making the agent's reasoning independently auditable. Users receive instant notifications via a Telegram bot, and all signals are visible through a public React dashboard.

The project targets the **AI Alpha & Data** track of the Turing Test 2026 Hackathon (sponsored by Mirana Ventures), with simultaneous bids for Grand Champion and Best UI/UX.

## Architecture Overview

```
[Mantle Blockchain]
      │
      ▼  web3.py polling every 30s
[collector.py]  ──────────────────────────────────────────────────┐
      │                                                            │
      ▼                                                            │
[ml_engine.py]  ←  IsolationForest + KMeans + SmartMoney Score   │
      │                                                            │
      ▼                                                            │
[signal_writer.py]  ──▶  MongliSignals.sol  (Mantle on-chain)     │
      │                                                            │
      ▼                                                            │
[Alert Dispatcher]                                                 │
      ├──▶ Telegram Bot   (real-time alerts to subscribers)       │
      └──▶ FastAPI REST   ──▶ React Dashboard (public)  ◀─────────┘
```

| Layer | Technology |
|---|---|
| Data collection | Python 3.11 + web3.py |
| ML / AI | scikit-learn (IsolationForest, KMeans), numpy, pandas |
| On-chain signals | Solidity 0.8.x + Hardhat on Mantle |
| Alerts | python-telegram-bot |
| API | FastAPI + APScheduler |
| Dashboard | React + Vite + Tailwind CSS + Recharts |
| Deploy | Vercel (frontend) · Mantle Mainnet (contract) |

## How the AI works

The agent runs three ML models in a pipeline over normalized on-chain data:

1. **IsolationForest** (`contamination=0.05`) — identifies wallets whose transaction patterns are statistical outliers in volume, frequency, or token diversity.
2. **KMeans** (`n_clusters=5`) — classifies wallets into behavioral archetypes: `retail`, `whale`, `bot`, `smart_money`, `new`.
3. **SmartMoney Score** — a hybrid rule+ML scorer that flags wallets accumulating assets *before* measurable price movements, scored on timing, position size, diversification, and profit history.

When confidence ≥ 70%, the signal is written on-chain via `recordSignal()`. The `dataHash` field contains a keccak256 of the full off-chain analysis object, allowing anyone to independently verify the reasoning.

## Setup Instructions

### Prerequisites

- Python 3.11+
- Node.js 18+
- A Mantle wallet with testnet MNT (faucet: https://faucet.sepolia.mantle.xyz)

### 1 — Clone and configure

```bash
git clone https://github.com/YOUR_USERNAME/Mongli_Agent_IA.git
cd Mongli_Agent_IA
cp .env.example .env
# Fill in your keys in .env
```

### 2 — Deploy the contract (testnet first)

```bash
cd contracts
npm install
# Add DEPLOYER_PRIVATE_KEY to ../.env first
npx hardhat run scripts/deploy.js --network mantleTestnet
# Script prints CONTRACT_ADDRESS — copy it to .env
# Script auto-verifies on Mantle Sepolia explorer
```

**Testnet faucet:** https://faucet.sepolia.mantle.xyz

### 3 — Run the agent

```bash
cd agent
pip install -r requirements.txt
python main.py
```

### 4 — Run the dashboard locally

```bash
cd frontend
npm install
npm run dev
```
